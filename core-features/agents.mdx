### Specialist Delegation

Agents route tasks to specialized domain experts in real-time:

- **Macro Analyst**: Monitors global liquidity, Fed policy, and geopolitical shocks.
- **Market State Analyst**: Classifies the regime (Risk-On/Off) to define current strategy.
- **Microstructure Analyst**: Identifies liquidation clusters and funding skews.
- **Trenches Analyst**: Scours social sentiment for micro-cap tokens (Rank 500+).
