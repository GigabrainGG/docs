Gigabrain is your AI trading desk for crypto. It has two core products: **Gigabrain Intel** for market intelligence and **Gigabrain Agents** for autonomous trading.

### What You Can Query

The Brain gives you access to 30+ data categories:

| Category | Examples |
| --- | --- |
| **Market Data** | Token prices, funding rates, open interest, liquidations, order book depth |
| **Onchain** | TVL, protocol revenue, DEX volumes, whale activity, exchange flows, bridge flows |
| **Macro** | ETF flows, options data, mining data, macro events calendar, correlations |
| **Derivatives** | Short squeeze candidates, perp DEX rankings, top gainers/losers |
| **Research** | Yield opportunities, token unlocks, L2 metrics, NFT collections, relative strength |
| **Sentiment** | Fear & Greed, narrative tracking, Polymarket predictions, portfolio analysis |

All data types support **structured JSON responses** when you ask for it. Add "Respond as JSON" to your query. See the [Brain API Overview](/developers/brain-api-overview) for the full reference.

## Gigabrain Agents

Agents are the execution side. They are autonomous trading programs that use The Brain to analyze markets and execute trades on your behalf, 24/7.

Each agent gets its own dedicated **Hyperliquid wallet**. You define the strategy (when to enter, how much to risk, when to exit), fund the wallet, and the agent handles the rest. It monitors markets on a schedule you set, analyzes conditions using The Brain, and trades when your criteria are met.
