# Builder Codes Guide

Learn about Gigabrain builder codes and how they provide benefits to both users and code creators.

## What are Builder Codes?

Builder codes are referral codes that provide benefits to both the code creator and the users who apply them. They're part of Gigabrain's community growth program.

## Benefits

### For Code Users

When you use a builder code:

- **Fee Discounts**: Reduced trading fees (up to 20% off)
- **Bonus Tokens**: Receive bonus tokens on first deposit
- **Premium Trial**: Extended trial of premium features
- **Priority Support**: Faster support response times

### For Code Creators

When others use your code:

- **Revenue Share**: Earn percentage of trading fees
- **Token Rewards**: Receive tokens based on referral activity
- **Community Recognition**: Build your reputation
- **Exclusive Benefits**: Access to creator-only features

## How to Use a Builder Code

### During Sign-Up

1. Visit [gigabrain.gg/login](https://gigabrain.gg/login)
2. Click **Sign Up**
3. Enter your details
4. Look for **Builder Code** field
5. Enter the code (e.g., `BUILDER123`)
6. Complete registration
7. Benefits activate automatically

### After Sign-Up

If you already have an account:

1. Log into Gigabrain Terminal
2. Go to **Settings** → **Account**
3. Find **Builder Code** section
4. Enter code
5. Click **Apply**

**Note**: Codes can only be applied once and cannot be changed later.

## How to Get a Builder Code

### Become a Builder

Requirements:

- Active Gigabrain user
- Minimum token holdings or trading volume
- Community participation
- Application approval

### Application Process

1. Visit [Gigabrain Terminal](https://gigabrain.gg/login)
2. Navigate to **Builder Program**
3. Click **Apply to be a Builder**
4. Fill out application:
   - Your background
   - Community involvement
   - Marketing plan
   - Social media presence
5. Submit for review
6. Wait for approval (typically 3-5 days)

### Code Creation

Once approved:

1. Choose your unique code (6-12 characters)
2. Set up your builder profile
3. Get your tracking dashboard
4. Start sharing your code

## FAQs

**Can I have multiple codes?**
No, each builder gets one unique code.

**Can I change my code?**
No, codes are permanent once created.

**Do codes expire?**
No, builder codes never expire unless violated terms.

**What if someone doesn't use my code?**
You won't receive benefits for that user.

**Can I use my own code?**
No, self-referrals are not allowed.
