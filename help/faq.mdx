# Frequently Asked Questions

Find answers to common questions about Gigabrain Terminal and services.

## General Questions

### What is Gigabrain?

Gigabrain is an AI-powered trading terminal that provides real-time market analysis, automated trading strategies, and comprehensive analytics for cryptocurrency traders.

### Who is Gigabrain for?

Gigabrain is designed for:

- Beginner traders looking to learn
- Experienced traders seeking AI assistance
- Institutional investors needing advanced tools
- Anyone interested in crypto trading

### Is Gigabrain free to use?

We offer a free tier with basic features. Premium features require a subscription or token holding. See our [Revenue Model](/token/revenue) for details.

## Account & Security

### How do I create an account?

Visit [gigabrain.gg/login](https://gigabrain.gg/login) and sign up with your email or connect your wallet.

### Is my data secure?

Yes. We use enterprise-grade encryption, secure API key storage, and never request withdrawal permissions from your exchange accounts.

### Do you have access to my funds?

No. Your funds remain in your exchange account or wallet. We only execute trades via API with trading permissions only.

### What is two-factor authentication (2FA)?

2FA adds an extra security layer. We highly recommend enabling it in your account settings.

## Trading

### Which exchanges are supported?

Currently supported:

- Binance
- Coinbase
- Kraken
- Bybit
- OKX

More exchanges are being added regularly.

### How accurate are the AI predictions?

Our AI models achieve 60-70% accuracy on average. However, no prediction is guaranteed, and all trading involves risk.

### Can I customize trading strategies?

Yes! Pro and Enterprise users can customize parameters, create custom bots, and adjust risk settings.

### What is the minimum investment?

- Test Fund: 100 USDC
- Alpha Fund: 10,000 USDC
- Auto-Trade: No minimum (depends on exchange requirements)

## Funds

- **Alpha Fund**: High-performance, higher minimum (10,000 USDC), advanced strategies

### Are the funds audited?

Yes, all funds undergo regular third-party audits and on-chain verification.

## Technical

### What is an API key?

An API key allows Gigabrain to execute trades on your behalf on connected exchanges. We only request trading permissions, never withdrawal permissions.

### How do I connect my exchange?

Follow our [Terminal User Guide](/help/terminal-user-guide) for step-by-step instructions.

### What if I lose my API key?

You can regenerate API keys in your exchange account settings. Update the new key in Gigabrain Terminal.

### Do you offer API access?

Yes, Pro and Enterprise users get API access. See [Developers](/developers/introduction) for documentation.
