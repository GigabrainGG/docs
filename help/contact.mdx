## What to Include

### When Contacting Support

Please provide:

1. **Your Account Email**: Help us locate your account
2. **Issue Description**: Detailed explanation of the problem
3. **Screenshots**: If applicable
4. **Steps to Reproduce**: What led to the issue
5. **Device/Browser**: Technical details
6. **Transaction IDs**: For trading-related issues

### Example Support Request

```
Subject: Unable to connect Binance API

Account Email: user@example.com
Issue: Getting "Invalid API Key" error when connecting Binance
Steps Taken:
1. Generated API key on Binance
2. Enabled trading permissions only
3. Copied key and secret to Gigabrain
4. Clicked Connect

Screenshot: [attached]
Browser: Chrome v120
```
